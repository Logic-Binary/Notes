 标识目标文件类型。

| 名称        | 值      | 意义      |
| --------- | ------ | ------- |
| ET_NONE   | 0      | 无文件类型   |
| ET_REL    | 1      | 可重定位文件  |
| ET_EXEC   | 2      | 可执行文件   |
| ET_DYN    | 3      | 共享目标文件  |
| ET_CORE   | 4      | 核心转储文件  |
| ET_LOPROC | 0xff00 | 处理器指定下限 |
| ET_HIPROC | 0xffff | 处理器指定上限 |

虽然核心转储文件的内容没有被详细说明，但 `ET_CORE` 还是被保留用于标志此类文件。从 `ET_LOPROC` 到 `ET_HIPROC` (包括边界)被保留用于处理器指定的场景。其它值在未来必要时可被赋予新的目标文件类型
